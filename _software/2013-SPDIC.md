This report describes the development of a single user computational tool designed to store and manage information related to the processes and services of a Reference Center. The system is implemented as a desktop application running on Windows 7 Ultimate, using MySQL as the database management system and Java as the programming language, with a graphical interface built using Swing components in the NetBeans IDE. The application supports comprehensive management of library resources, including collections, users, and services such as loans, references, and document copying, as well as the administration of library activities, publications, and report generation. The main project's result is the design and implementation of an integrated information system aimed at improving the organization, control, and accessibility of reference center data.
